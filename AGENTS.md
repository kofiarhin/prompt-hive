## Tech Stack

### Frontend

- React (Vite)
- JavaScript (default)
- Tailwind CSS
- React Router
- TanStack Query (server state)
- Redux Toolkit (global client state only)
- Axios (shared HTTP client)
- Socket.IO (only when required)
- Vitest (testing)
- React Testing Library

### Backend

- Node.js
- Express
- MongoDB (Mongoose)
- Jest (testing)
- Supertest (API testing)
- Socket.IO (only when required)
